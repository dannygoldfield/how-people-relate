# Design Principles

Every creative decision should make it easier to see another person.

1. Reduce Stimulation
2. Invite Mentalizing
3. Favor Description
4. Prefer Restraint
5. Make Every Element Earn Its Place
6. Embrace Ambiguity
7. Let the Viewer Participate
8. Design for Replay
9. Build Systems
10. Measure Everything
11. Design for Sustainability
12. Automate Everything Possible
