# Entity Model

How People Relate is built from a small set of related entities.

1. Portrait

A single exported image.

↓

2. Recipe

A specification for generating candidate videos from a portrait.

↓

3. Candidate

A generated video created from a recipe.

↓

4. Selection

The chosen candidate.

↓

5. Episode

A published work.
